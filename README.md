# Приложение "Заметки" для Android

## Основные возможности

### Управление заметками
- **Создание заметок** с заголовком, описанием и запланированным временем
- **Редактирование** существующих заметок
- **Удаление** заметок
- **Отметка выполнения** задач

### Календарное планирование
- Выбор даты и времени для каждой заметки
- Удобные селекторы даты и времени
- Сортировка заметок по дате и приоритету

### Уведомления
- Автоматические напоминания за **10 минут** до запланированного времени
- Уведомления отменяются при выполнении или удалении задачи

### Категории и организация
- Создание и выбор категорий для заметок
- Фильтрация заметок по категориям
- Автоматическое отслеживание всех используемых категорий

### Приоритеты
- Три уровня приоритета: Низкий, Средний, Высокий
- Цветовая индикация приоритета
- Сортировка по приоритету

### Поиск
- Быстрый поиск по заголовку и описанию
- Фильтрация в реальном времени

### Статистика
- Количество активных заметок
- Количество выполненных заметок
- Процент выполнения задач

### Выполненные задачи
- Отдельный просмотр выполненных задач
- Переключение между активными и выполненными заметками

### Разрешения

Приложение запрашивает следующие разрешения:
- `POST_NOTIFICATIONS` - для отправки уведомлений (Android 13+)
- `SCHEDULE_EXACT_ALARM` - для точного планирования напоминаний
- `USE_EXACT_ALARM` - для использования точных будильников

## Использование

### Создание заметки
1. Нажмите кнопку **"+"** в правом нижнем углу
2. Заполните заголовок и описание
3. Выберите категорию (или создайте новую)
4. Установите приоритет
5. Выберите дату и время
6. Нажмите **"Сохранить"**

### Редактирование заметки
1. Нажмите на карточку заметки
2. Внесите изменения
3. Нажмите **"Сохранить"**

### Поиск заметок
1. Введите текст в поле поиска вверху экрана
2. Результаты фильтруются автоматически

### Фильтрация по категориям
1. Используйте чипы категорий под полем поиска
2. Нажмите на категорию для фильтрации
3. Нажмите "Все" для сброса фильтра

### Просмотр статистики
1. Нажмите иконку статистики в верхней панели
2. Просмотрите количество активных и выполненных заметок

### Просмотр выполненных задач
1. Нажмите иконку галочки в верхней панели
2. Переключитесь между активными и выполненными заметками

## Технологии

- **Kotlin** - язык программирования
- **Jetpack Compose** - современный UI фреймворк
- **Room** - локальная база данных
- **WorkManager** - планирование фоновых задач
- **Material Design 3** - дизайн-система
- **MVVM** - архитектурный паттерн

## Совместимость

- **Минимальная версия Android:** 7.0 (API 24)
- **Целевая версия Android:** 16.0 (API 36)
- **Рекомендуемая версия:** Android 10.0 (API 29) или выше
